Testing DuckSoup in an iframe

One ore more URI query parameters are missing or incorrect:

  • room (room name)
  • uid (user unique id)
  • name (user display name)
  • proc (ask for audio/video processing, 0 ou 1)
  • duration (conversation duration in seconds)

Examples

  • /test_embed/?room=hall&uid=1&name=zoe&proc=0&duration=30
  • /test_embed/?room=hall&uid=2&name=hal&proc=0&duration=30

You may specify optional parameters:

  • videoCodec (vp8 - default -, h264 or vp9)
  • size (room size)
