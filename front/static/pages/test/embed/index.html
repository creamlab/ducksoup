Testing DuckSoup in an iframe

One ore more URI query parameters are missing or incorrect:

  • room (room name)
  • userId (user unique id)
  • duration (conversation duration in seconds)

Examples

  • /test/embed/?roomId=hall&userId=1&duration=30
  • /test/embed/?roomId=hall&userId=2&duration=30

You may specify optional parameters:

  • videoCodec (VP8 - default - or H264)
  • size (room size)
